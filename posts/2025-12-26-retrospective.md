## 「去年までの振り返り」の振り返り

2024年は「2025年はコミュニティ活動に精を出したい」ということを言っていました。しかし、その気持はすっかり薄れてしまいました。言い換えると、「所属先の企業がコミュニティ」という気分になりました。ここには尊敬する同僚も多いし、この場所で切磋琢磨することの方が優先度が高いと思うようになりました。

使っているツールも基本的に2024年と大差なく、相変わらずメモ書きは[Obsidian](https://obsidian.md/)で、ソーシャルメディアは[Bluesky](https://bsky.app/)で、という感じです。ペアプログラミングは[Tuple](https://tuple.app/)で、ランチャーには[Raycast](https://www.raycast.com/)を使っています。あ、パスワードマネージャに1Passwordを使い始めました。転職先が1Passwordを配布していて、[Businessアカウントが無料で使える](https://blog.jxck.io/entries/2025-11-09/family-paln.html)と知ったからです。1PasswordでSSH鍵や環境変数の管理をするようになり、手元のマシンから秘密情報をほぼ一掃できました。

あとは[IntelliJ](https://www.jetbrains.com/idea/)はもう使わないかもしれないと思ったけど、色々あってまだ現役です。でも[Kotlin](https://kotlinlang.org/)は書いていないです。IntelliJ、全体的な出来はとてもいいけど、GitHub Copilotプラグインを使おうとすると粗が目立つ、というのが今の感想です。

## 2025年のできごと

### 転職

所属していたチームがレイオフで丸ごとなくなり、転職先を探す羽目になりました。たまたまポジションが空いていた外資系企業に転職しました。

新しい職場では、まだ8~9ヶ月程度しか過ごしていないですが、すでに「色々あったな」という気持ちです。率直に言えば「つらい」経験もあり、今後どうしたものかと思います。今は尊敬できる素晴らしいチームメイトに支えられて「何とかやれている」という感じです。

どの企業もそうだけど、良いところと悪いところがあり、悪いところが極まると「つらい」状態になります。このままだと2026年もどこかのタイミングでつらい状態になりそうです。「わかってるなら何とかするべき」という話はもっともで、そうあるべきだと思います。しかし、そこの是正にモチベーションが持てないのは、ポジションのミスマッチを感じていることから来ていそうです。なので、ポジションの方を「何とかしたい」というのを2026年前半の目標にしたいです。

### 英語

2011年にカナダに引越して現地企業で働き、2015年に帰国しました。その後も英語環境で揉まれてきています。とはいえ、帰国してから、上司が全く日本語を話せない環境は初めてなので、英語をブラッシュアップしようかと思いました。

ブラッシュアップするにも、自分のベンチマークを取るのが先だろうと思い、英検1級を取ろうと思いました。しかし、夏頃に試験を受けた結果は不合格でした。英検を侮っていました。自分の周囲で同じようなキャリアを歩んでいる友人らは、英検1級をさしたる苦もなく取得しているように見えていました。残念ながら、自分にはそのような実力がないという事実を突きつけられました。

英検1級の合格は2026年に持ち越しの目標となりました。古典的だけど、[Anki](https://ankiweb.net/)を使って語彙を増やす訓練を今はしています。来年の夏くらいに試験を受けようと思っており、その直前には難しめの長文を読む練習もしようと思っています。

### 子育て

難しさを感じた一年でした。

長男が発達障害児で、次男が定型発達です。長男とのコミュニケーションは難しいけど、次男は諦めずに長男に話しかけてくれるので、ありがたく思っていました。しかし、次男は長男の影響を強く受けてしまいました。彼もぶつぶつ独り言を話したり、奇声を上げるようになってしまいました。

結果として、次男も幼稚園で友だちを作るのに苦労していて、今後どうしたものかという感じです。

次男の知性が高い分、彼が長男を煽って喧嘩になってしまうことも多々あります。結果として兄弟を一緒に扱うのが難しく、別々に行動させることも増えてしまいました。

2026年も引き続き子育てに悩むことでしょう。

### 健康

#### 右膝痛

2015年にジョギング中に右膝を痛めてから、ずっと右膝に「重さ」を感じていました。とはいえ、生活に支障はないので、特に病院にもかからず過ごしていました。これが2025年の秋くらいから「痛み」を感じるレベルになったので、重い腰をあげて整形外科にかかりました。

近所の整形外科では、レントゲンを撮られて「問題ないですね」と帰されてしまいました。ちょっと消沈しましたが、痛いものは痛いので、MRIのある病院にセカンドオピニオンを求めに行きました。

MRIを撮ってもらった結果も「基本的には問題なし」という扱いで、強いていえばちょっと腫れてるかも、くらいでした。湿布をもらって様子見、というステータスです。まだ劇的な効果は感じられていなくて、来年も痛むようであれば次のステップを検討する、というステータスです。

### リモートワーク @ 軽井沢

2024年に続いて、2025年も夏に軽井沢でリモートワークをしました。6月中旬から9月下旬くらいの期間です。

家族が冷房に弱いので、冷房がなくても過ごせる場所に行きたい、というのがモチベーションでした。ただ、二年の経験を経て「軽井沢でのワーケーションはきつい」という結論にいたりました。

軽井沢は確かに涼しいです。しかし、仕事をするには不便でした。まず、インターネット回線が貧弱です。選んだ物件は「モバイルルーター利用できる」という触れ込みでしたが、遅い上に安定しないのでオンラインミーティングが困難でした。Povoでテザリングをしてみたものの、これも安定せず、地方のセルラーネットワークの厳しさを感じました。なお、[風越公園](https://www.kazakoshi-park.jp/)の近くの物件に住んでいたのですが、ここでイベントをしているときは顕著にネットワークが遅くなりました。

電源も安定せず、何度も停電しました。停電の復旧に4~5時間かかることもあり、ろうそくを灯して夜を過ごすこともありました。後で知ったのですが、現地では自家発電設備を持っているご家庭も多いそうです。

細かいことを言うと、仕事用に椅子や机を準備しなかったため、長時間の仕事で体が痛んだり、仕事用の個室がないため子どもたちに気軽に邪魔されてしまうという問題もありました。

東京の良さを再確認しました。2026年は軽井沢でのリモートワークはやらないと思います。

### 趣味

#### B'z

2025年はB'zのライブに二度行きました。UNITEとFYOPです。どちらも良かった。

UNITEはOne Ok Rockとの対バンの日程に当選しました。One Ok Rockは未知のバンドだったので、当選後はOne Ok Rockのヒット曲を意識的に聴いてライブに備えました。事前の感想は「結構いいじゃん」程度でしたが、ライブに行ってからは「かなりいいじゃん」に変わりました。「完全感覚ドリーマー」や、B'zの稲葉さんが参加した「The Beginning」は結構お気に入りです。

ところで、UNITEはB'z松本さんが体調不良で欠場だったため、いつもとは様子が違いました。実際にはアンコール曲だけ登場して演奏をしてくれましたが、彼は明らかにいつもより痩せてしまっており、無理を押している感じが伝わりました。

FYOPツアーは東京ドームの公演に参加しました。語彙に乏しいですが「とても良かった」です。楽曲以外にもMCにぐっと来ました。細かいところは間違えているかもしれませんが、

> Follow Your Own Passionというタイトルでツアーをやってきました。何事も最初はパッションに突き動かされて進むわけだけど、やっていくと選択肢が増えてきて、雑念も増えてきて、迷いが生じることもあると思います。そういうとき、自分のパッションが何なのか、そこに立ち返ることが大事だと思っています。

...という旨のことを言っていました。まさに自分がパッションを無視して生きているように感じていたので、響く言葉でした。

松本さんもMCで話していて、彼は「（ギターを弾くという、自分にとって）当然のことが、普通にできて嬉しい」という旨の発言をしていました。これはUNITEのときの体調不良から来ている言葉でしょう。自分はまだ大病を患ったことはないけど、彼のようなキャリアの持ち主からの言葉だと思うと重みを感じます。

楽曲としては、東京、声明、Still Aliveあたりが特に心に響きました。

#### 星のカービィ

次男が「星のカービィ」に興味を持ったので、いくつかNintendo Switchのタイトルをプレイしました。具体的には「星のカービィ Wii デラックス」、「星のカービィ ディスカバリー」、「星のカービィ スターアライズ」あたりです。

星のカービィ、僕が小学生の頃からずっと第一線で人気があってすごいなあと思いました。そこだけ切り取るとB'zみたいだ...。しかも、よく考えると、吸い込んで飲み込んで、という演出は残酷な気がするけど、流血表現も一切なくて、ゲームとして楽しいというのがすごい、という感想です。カービィ、とてもかわいい。

#### 資格

毎年ちょっとずつ資格を取っていて、資格取得が趣味になりつつあります。自分は資格取得を通じた勉強の仕方が得意かもしれないとも思います。

今年はGitHubに関する資格を取りました。具体的にはGitHub Foundations, GitHub Administration, GitHub Advanced Security, GitHub Actions, GitHub Copilotの5種類です。現時点でGitHubが提供している技術系の資格はすべて取りました。

これらは仕事で役立てたいというモチベーションで取ったのですが、2026年は自分の興味関心にもとづいたものを狙っていこうと思います。

#### 3Dプリンタ

新しい趣味として3Dプリンタを購入しました。Bambu Lab A1 miniです。まったくの初心者ですが、すでに印刷を楽しく感じています。パズル、タオルかけ、タブレットスタンド、フィジェットなどを作ってきました。もっと活用したいです。

現状は自分でモデリングしていないし、部品の名前やスペックなどもわからないことが多いです。伸びしろだらけです。なお、3Dプリンタを稼働させていない時間を「もったいない」と思うような感覚が出てきたので、沼の一歩手前だな...という気持ちもあります。

#### 漫画

過去に読んだ漫画を再読していて、いかにも脳が老化している感じです。「ナルト」とか「ヒカルの碁」とか、今読んでも面白い。

新しめのタイトルだと「カモのネギには毒がある」や「ウィッチウォッチ」を楽しく読んでいます。どちらも著者の過去作が好きで、すっと手に取れました。

ジャンプ関連の本はゼブラックで読んでいて、一話ごとに漫画本のレコメンドがあるけど、この推薦結果が自分にヒットすることがほとんどなくて、マンガ本のレコメンドエンジンは伸びしろがある市場なのではないか、と思っています。自分がその市場に飛び込む可能性は低いけど。
